## このプロジェクトについて
Rampは「勉強できない人を助けるアプリ」です。  
勉強の極意の元となったアプリで、僕が初めて作ったアプリです。
- 使用技術：Swift, SwiftUI
- 開発期間：半年  
- 担当　　：全部
